本地部署打开网页

html或php文件本地打开网页方式：

  1. 用VScode打开文件所在文件夹，再打开文件用php server：serve project

  2. 用phpstudy（小皮）：首先文件需要在网站根目录下（查询方式：软件-网站-管理-打开根目录）；再打开浏览器在网址栏输入127.0.0.1/【根目录下的文件路径】

    如若直接保存在根目录下则：127.0.0.1/php.php

    若在根目录下的文件夹里：127.0.0.1/test/php.php

总理解

php(Hypertext Preprocessor 超文本处理器):后端编程语言，主要作用是生成html文档

其它：Apache作为网站服务器软件，负责接收浏览器的访问请求（如网页、图片等资源请求），在服务器上找到对应文件（如HTML、PHP页面），并将这些文件的内容发送回用户的浏览器进行显示，同时支持处理动态网页（如配合PHP执行程序逻辑）。简言之，它是连接用户浏览器与网站文件的中介，让网页能在互联网上被访问。

关于 '' 与 " " :

  • 内容有变量或转义特殊字符（/n），或字符串包含单引号时：用 “ “
  • 字符（串）用 '' 或 " " ；字符串拼接加. (可”字符串”. “<br> “实现换行)
  • " " 自动解析变量、转义字符； '' 不解析，原样输出。
  • 数字、布尔值、变量名本身不加引号 ；

当 PHP 解析器遇到结束标记 ?> 时，会立即停止解析，并将后续的所有内容（包括空格、换行符等）视为普通文本直接输出到响应流中。—>若文件内容仅仅包含 PHP 代码，最好在文件末尾删除 PHP 结束标记

$_SERVER

是一个包含了header、path、script locations(脚本位置)等信息的数组。

https://www.runoob.com/php/php-superglobals.html

元素 值
$_SERVER[‘PHP_SELF’] 当前执行脚本的文件名，与 document root 有关
$_SERVER[‘SERVER_ADDR’] 运行脚本所在的服务器的 IP 地址
$_SERVER[‘REQUEST_METHOD’] 访问页面使用的请求方法，如，”GET”, “HEAD”，”POST”，”PUT”
$_SERVER[‘QUERY_STRING’] 查询字符串（如果有的话，通过它进行页面访问）
$_SERVER[‘HTTP_HOST’] 当前请求头中 Host: 项的内容
$_SERVER[‘HTTP_REFERER’] 引导用户代理到当前页的前一页的地址（不可信）
$_SERVER[‘REMOTE_ADDR’] 浏览当前页面的用户的 IP 地址
$_SERVER[‘REMOTE_HOST’] 浏览当前页面的用户的主机名
$_SERVER[‘SCRIPT_FILENAME’] 当前执行脚本的绝对路径

$_POST

用于收集HTML表单提交的数据

在HTML form标签指定属性：”method=”post”。

以 POST 方法发送的信息，对任何人都不可见，发送信息的量没有限制。

一个输入字段（input）及提交按钮(submit)的表单(form)。 当用户点击 “Submit” 按钮提交表单数据时, 表单数据将发送至 < form > 标签中 action 属性中指定的脚本文件。 使用 $_REQUEST 或 $_POST 来收集表单中的 input 字段数据。

$_GET

用于收集表单数据（ 以 get 方式发送的请求，及浏览器地址栏中的 ? 之后的内容 ），在HTML form标签的指定该属性：method=”get”

$_GET 也可以收集URL中发送的数据

特点： 从以 GET 方法的表单发送的信息， 所有的变量名和值会显示在浏览器的地址栏， 变量值不能超过 2000 个字符。

$_REQUEST

post 和 get 方式发送过来的请求都可以接受 ， get 会显示在 url 中（有字符数限制），post 不会在 url 中显示，可以传递任意多的数据

抽象类

有一个方法为抽象的【 只能声明调用方式（参数），不能定义具体的功能实现 】，则此类必须被声明为抽象类，且不能被实例化。

子类继承时必须以相同或更宽松的访问控制定义父类中的所有抽象方法 ，子类方法中的可选参数可以比父类抽象方法中多

Static

static的类属性或者方法可不实例化类直接访问，可通过静态方法访问，可由self::$n_static 或class_name::$n_static 访问；不能用$this访问,不能通过一个类已实例化的对来访问 ，不可以由对象 -> 属性(或方法) 访问

Final

子类无法覆盖被声明为final的方法

被声明为final的类不能被继承
