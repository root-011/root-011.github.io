语法

变量与常量

  • 常量：
    • var：全局作用域
    • let：声明时可不初始化；可被修改
    • const：声明时必须初始化数值；不可被修改，但若声明的是数组或对象，可对其内部元素进行修改

运算符

== 为不考虑数值类型的等

===为考虑数值类型的等

三目运算符：

数据类型

  • String：用"" 或 '' ;如x = "Reese";
  • Number：包括小数与整数
  • Boolean：true 或 false
  • null：值为空；如y = null ;
  • undefined：值未定义（与null区分）；let y = undefined ; 等价于 let y;
